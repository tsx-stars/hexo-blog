# var、let、const

let 声明变量，const 为常量，常量就是不能被修改
let 和 const 声明变量不存在变量提升，如果要使用这个变量，我们需要在变量定义之后使用；
let 和 const 不能重复声明变量，如果重复声明会报错；
用 let 和 const 在全局声明变量不会给 window 增加属性；
let 和 const 出现在代码块中，会把代码块 (字面量声明对象除外) 变成块级作用域，并且出现暂时性死区

# es6 的 class

在 es5 中，我们用大小写来区分普通函数和构造函数，这种方式给开发者的感觉不够严谨，所以在 es6 中换成了 class，有这个单词就代表是个构造函数
class 写法只是让对象原型的写法更加清晰跟构造函数类似
先定义一个类，里面有个 constructor 方法，这就是构造方法，之前写在构造函数内的属性就写在里面
constructor 之外写构造方法，并且不需要加 function

# promise

有了 Promise 对象，可以将异步操作以同步的流程表达出来，避免了层层嵌套的回调函数，它本质上还是利用了回调函数
resolv 对应成功时的状态，成功后调用 then 方法
reject 对应失败时的状态失败调用 catch 方法
成功失败都调用 finally
es7 引入了 async/awiat
async 告诉程序这是一个异步操作，一般写在 function 前面，await 是一个操作符，即 await 后面是一个 promise 函数

# symbol 有什么用处

可以用来表示一个独一无二的变量防止命名冲突。
还可以利用 symbol 不会被常规的方法（除了 Object.getOwnPropertySymbols 外）遍历到，所以可以用来模拟私有变量

# 箭头函数和普通函数有啥区别

普通函数通过 function 关键字定义，
this 指向不同，箭头函数没有自己的 this,this 是定义时所在的对象，普通函数看前面有没有., 点前面是谁 this 就是谁，没有。就是 window
箭头函数是匿名函数，不能作为构造函数，不能使用 new, 当使用 new 进行函数调用时会报错。
箭头函数没有原型属性
箭头函数没有 super, 没有 arguments, 没有 new.target, 没有 [[Construct]] 方法

# 数组新增的方法

isArray()
find () 和 findIndex ()
Array.from()
includes()
Array.of () // 用于将一组值，转换为数组
copyWithin () // 在当前数组内部，将指定位置的成员复制到其他位置
fill () // 方法使用给定值，填充一个数组，fill 方法还可以接受第二个和第三个参数，用于指定填充的起始位置和结束位置
entries ()，keys () 和 values () // 返回一个遍历器 (iterable) 对象
flat (deep=1) // 将嵌套的数组 “拉平”，多维数组变一维
flatMap () // 先 map 再对结果 flat
